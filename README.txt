El siguiente proyecto se realiza para la materia Estructuras de Datos de la Universidad Nacional de Colombia, y consiste en una aplicación para la gestión de libros de Biblioteca Infantil, enfocado a que los niños pequeños tengan un acceso más fácil e intuitivo al catálogo de una biblioteca.
